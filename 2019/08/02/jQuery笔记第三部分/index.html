jQuery笔记第三部分

发表于 2019-08-02 | 分类于 jQuery
增加样式.addClass 移除样式.removeClass

鼠标事件/键盘事件

click()单击 mouseover() mouseout() keydown() keyup() keypress() event.keyCode按键ASC2码 focus() blur()

绑定事件的方式

.bind(“click”,function()) 或者.on({json}) .unbind(“click”,function())移除事件 .unbind()移除所有
.houver(function,function)鼠标经过离开
.toggle()连续点击(1.9版本后删除)
$(form).submit提交
.addClass(‘name name’)

内容操作

$(<>text<>)创建节点
.html() .text() .val() .attr()属性获取 .attt(“”,””)属性修改

节点操作

$().append() .appendTo($())添加至后面 $().prepend() .prependTo($())添加至前面
.after元素后插入
.remove() .empty() 删除/清空
复制节点 .clone()复制节点 .clone(true)复制节点与事件
替换节点 $().replaceWith() .replaceAll($())颠倒

节点遍历

.children()子元素 .parent()前辈元素 .parents 祖先元素
.next()紧邻元素之后 .prev()紧邻元素之前 .sibings()同胞元素
工具函数$.each($(),function(index,item))
成员函数$().each(function(index,item))

动画

.show(time,function)显示 .hidde(time,function)隐藏
.fadeOut()淡出 。fadeIn淡入
.slideDown()下滑 .slideUp()上滑
.animate({json},time,function)
+= 原来位置偏移
.find(选择器)内部查找 .filter()过滤

cookie

$.cookie(key:value{expires:7}) $.removeCookie(‘’)
