jQuery笔记第一部分

发表于 2019-07-30 | 分类于 jQuery

看了眼之前的笔记，发现一个误区，我写成了教程，这次我尝试更精简的总结

jQuery入门

$(选择器)是一个数组
ready()方法表示文档就绪，类似于onload，$(document.ready())可以简写成$()，一个页面可以同时拥有多个
css()方法
换名字 var name = jQuery.noConflict()
jQuery = $
dom.value = jQuery.val()
dom.innerhtml = jQuery.html()
jq对象是数组
jq隐式遍历
jq.get(index)
设置与读取同一个方法.html(‘’)
样式.css(‘color’,’red’)
设置value .val(‘’)
宽高可以使用css，也可以.width(‘’).height(‘’)
连用方法 .html(‘’).css(‘’)
.css({‘’:’’,’’:’’}) json对象

基本选择器

$(#id),$(.class),$(#id,.class),$(a.b),$(*)

层次选择器

$(div#first div) $(div#first > div) $(div#first + div)相邻 $(div#first ~ div)后全部

过滤选择器

$(span:first)第一个span :last :not(.class) :even偶数 :odd奇数

css进阶

透明 opacity:value(0-1|继承)，影响字体颜色
rgba a(0-1)，不影响字体颜色
